# Robotic Kitchen

> Smart Electronics, Robotics & Automation Solutions

Robotic Kitchen is a modern, premium, futuristic, and fully responsive e-commerce front-end application tailored for robotics enthusiasts, students, makers, and engineers. It provides a platform to explore components, buy electronics, and learn through step-by-step DIY project tutorials.

![Robotic Kitchen Logo Concept](public/vite.svg)

## Features

- **Futuristic UI/UX:** Built with a dark mode base, glassmorphism components, and neon blue/orange accents.
- **Responsive Design:** Fully responsive layout for Mobile, Tablet, and Desktop displays.
- **E-Commerce Shop:** A robust product grid featuring Arduino boards, sensors, motors, drone components, and more.
- **DIY Projects Hub:** Step-by-step tutorials with wiring diagrams, code snippets, and material lists (e.g., Quadcopter Drone Build).
- **Core Pages:** Home, Shop, Categories, Projects, Blog, Cart, and Login placeholders.

## Technology Stack

- **Frontend Framework:** React.js powered by Vite for blazing-fast builds.
- **Routing:** React Router DOM for seamless Single Page Application (SPA) navigation.
- **Styling:** Custom Vanilla CSS with robust CSS variables for theming.
- **Icons:** Lucide React for consistent, crisp SVG iconography.

## Getting Started

### Prerequisites

Make sure you have Node.js and npm installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/robotic-kitchen.git
   ```

2. Navigate to the project directory:
   ```bash
   cd "robotic-kitchen"
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and navigate to `http://localhost:5173`.
